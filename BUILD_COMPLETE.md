## 🚀 Quick Start (3 Steps)

### Step 1: Database (5 minutes)
```
1. Create Supabase project
2. Copy Project URL & Anon Key
3. Run /scripts/schema.sql in Supabase SQL Editor
4. Verify 11 tables created
```

### Step 2: Storage (2 minutes)
```
1. Enable Vercel Blob in your project
2. Copy Blob token
```

### Step 3: Deploy (3 minutes)
```
1. Add environment variables to Vercel:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - BLOB_READ_WRITE_TOKEN
   - JWT_SECRET
2. Redeploy
3. Create admin account (SQL UPDATE)
```

**Total: 10 minutes to production!**
